Reusable design system monorepo for the Artivism ecosystem.

## Installation

```bash
npm install @artivism/tokens @artivism/ui


## Quick Start

npm install
npm run dev

## Architecture

- @artivism/tokens: CSS design tokens
- @artivism/ui: React component library
- apps/design: Living documentation site

## Scripts

- npm run dev: Start living spec
- npm run build: Build all packages
- npm run changeset: Create version changeset

Living documentation: design.artivism.it
